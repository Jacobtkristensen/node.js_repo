Nodemon

Nodemon is a tool that helps develop Node.js applications by automatically restarting the server when changes are detected in the codebase. This can save time and effort during development by eliminating the need to manually stop and restart the server after each code change.

Nodemon should only be used in development. 


      //globally
      npm install nodemon -g 
      //locally
      npm install nodemon --save-dev

To start a Node.js application with Nodemon, use the following command:


        nodemon app.js

Loops

Different loop methods.
for-loops, map, filter, sort, find, reduce

for loop

only use this when finger counting is needed.


    for (let i = 0; i < numbers.length; i++) {
      console.log(numbers[i]);
    }

map

map creates a new array and applies a function to each element in the array.
In this case it multiplies each element by 2. 


    const numbers = [1, 2, 3, 4, 5];

    const doubledNumbers = numbers.map((number) => number * 2);
    // Result: [2, 4, 6, 8, 10]

filter

filter creates a new array with all elements that pass the % 2 === 0.


      const numbers = [1, 2, 3, 4, 5];

      const evenNumbers = numbers.filter((number) => number % 2 === 0);
      console.log(evenNumbers); 
      // Result: [2, 4]

sort

sort sorts the elements of an array in place and returns the sorted array.
respectively in increasing or decreasing order. 


      const numbers = [5, 3, 4, 1, 2];

      //increasing order
      numbers.sort((a, b) => a - b); 
      console.log(numbers); 
      // Result: [1, 2, 3, 4, 5]
      
      //decreasing order
      numbers.sort((a, b) => b - a); 
      console.log(numbers); 
      // Result: [5, 4, 3, 2, 1]

find

find returns the first element in the array that pass the % 2 === 0 condition.
if no element passes, it returns undefined. 


      const numbers = [1, 2, 3, 4, 5];

      const firstEvenNumber = numbers.find((number) => number % 2 === 0);
      console.log(firstEvenNumber); 
      // Result: 2

reduce

reduce applies a function to each element in an array and produces a single value.
in this case i sum all the elements in the array. 


      const numbers = [1, 2, 3, 4, 5];

      const sum = numbers.reduce((accumulator, currentValue) => accumulator + currentValue, 0);
      console.log(sum); 
      // Result: 15

Anatomy of an URL

using this 'https://www.google.dk/search?q=test&' as an example to break down.

Protocol: `https` - specifies protocol used, and indicates that the connection is secure.
Domain: `www.google.dk` can be broken down into:
Subdomain: `www`
Domain: `google`
Top-level domain: `dk`
Path: `/search`
Query string: `?q=test&` - contains key-value pairs separated by `&`. q is the key and test is the value.
Fragment: `#` - used to link to a specific section of a page. Not present in this example, but would be indicated by a `#` followed by the section name.
I have done this at the top of this page to quickly the different sections of the page.

Body parsing

Before we can begin working with the data sent in a POST request, we need to parse the body of the request. This is built into express. 


      const express = require("express");
      const app = express();

      //necessary for POST, PUT, PATCH, DELETE
      app.use(express.json());

POST request

Handling a POST request with express is simple. we use the above body-parser.
This example simply returns the body of the request to the client. 


      app.post("/postman", (req, res) => {
        console.log(req.body);
        res.send(req.body);
      });
